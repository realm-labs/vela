This page will document the practical hot reload workflow used by embedding hosts.

## Planned Coverage

- Compile the initial program version.
- Create a runtime from a hot reload version.
- Compile an update.
- Stage the update.
- Apply the update at a safe point.
- Inspect update and rejection reports.
